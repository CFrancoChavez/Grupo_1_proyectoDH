# Grupo_1_SPORTS-HOUSE / Proyecto de Node.js - Express - EJS - JavasCript - CSS - Sequelize - MySql
SPORTS HOUSE

Este trabajo tiene como objetivo la creación de un ecomerce en el marco del Proyecto Integrador en Digital House. 
Para poder interactuar con el mismo es necesario tener instalado XAMPP (es un paquete de software libre, que consiste principalmente en el sistema de gestión de bases de datos MySQL, el servidor web Apache y los intérpretes para lenguajes de script PHP y Perl).
En primer lugar se inicia dandole start al modulo Apache y MySql en este ultimo se hace click en el botón admin para poder acceder a la base de datos.
En este sentido se trabajo en una base de datos relacional vinculada al codigo con Sequelize.
Una vez ubicados en la carpeta ACTIVIDAD_SPORTSHOUSE abrimos la terminal y ejecutamos el comando npm start para levantar el servidor.
